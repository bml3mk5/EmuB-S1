  擬似漢字辞書ROM S1DIC.ROM を作成するプログラムです。

● 用意するもの

  スクリプト言語 perl または python

● 実行方法

  perlを使用する場合はmkdic.plを、pythonを使用する場合はmkdic.pyを実行して
  ください。

  成功すると S1DIC.ROM が生成されます。サイズは32KBです。
